## Use and Install on the Command Line

```bashh
git clone https://github.com/Josephrp/LablabAutogen.git
```

```bash
cd LablabAutogen
```

```bash
set PATH=%PATH%;%CD%
```

```bash
nano app.py
```

edit line 17 "    ```"openai_api_key": "YOUR_KEY_HERE",  # OpenAI API Key``` with your key

then press:

```nano
 control + x
```

Write :

```nano
Y
```

to save then type :

```bash
pip install -r requirements.txt
```

and finally :

```bash
python app.py
```
to run.
