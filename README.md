## Use and Install

```
git clone https://github.com/Josephrp/LablabAutogen.git
```

```
cd LablabAutogen
```

```
nano app.py
```

edit line 17 "    ```"openai_api_key": "YOUR_KEY_HERE",  # OpenAI API Key``` with your key

then press:

```
 control + x
```

Write :

```
Y
```

to save then type :

```
pip install -r requirements.txt
```

and finally :

```
python app.py
```
to run.
